Key Features

Supervised Learning

Uses SVM, Random Forest, and XGBoost to achieve high accuracy.

Real-World Dataset

Trained on SMS spam messages from the UCI Machine Learning Repository.

Easy to Use

Simply clone the repository, install dependencies, and run the model.

How It Works

1. Preprocessing

Text is cleaned, tokenized, and converted into TF-IDF features.

2. Model Training

Supervised models classify messages into spam or ham.

3. Evaluation

Models are tested for accuracy, precision, recall, and F1-score.

4. Predictions

The trained model predicts whether an SMS is spam or not.

Get Started

Follow these steps to run the project:

# Clone the repository
git clone https://github.com/your-username/your-repo.git
cd your-repo

# Install dependencies
